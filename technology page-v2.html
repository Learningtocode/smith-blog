Description of HTML

Hypertext Markup Language is the what makes up web sites on the Internet. It uses tags enclosed most of the time in angle brackets. It allows media such as images or videos to be embedded into a web site. HTML allows the use of paragraphs, titles, and headings to be shown on a web browser. For example the paragraphs on a web site are made by a paragraph tag of two p's enclosed in angle brackets with the words that show on the web site between the brackets. Wikipedia said "HTML markup consists of several key components, including tags (and their attributes), character-based data types, character references and entity references".

Description of CSS

Cascading Style Sheets is a style sheet language that we use to create style to web sites like the layout, colors, and fonts on web sites. It works with HTML and other XML documents to accomplish this goal. Majority of web sites use CSS for the presentation of their web sites. It deals with margins, padding, and borders so objects on paged are set right on the web page.

Description of Bootstrap

Bootstrap is literally a collection of tools you use for your web site. Wikipedia said "It contains HTML and CSS-based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions". When you go on a web site and see a slide show, that is one of Bootstraps tool. It is compatible will all the newest and was rated the number one project this year on GitHub.

Description of JavaScript

JavaScript is an object-oriented computer programming language commonly used to create interactive effects within web browsers. We are using JavaScript to make the game snake in this case. It does not interact with web servers what so ever but instead with web browsers as stated previously.

Description of MelonJS

MelonJS is a computer game program that adapts familiar aspects of Javascript. MelonJS integrates Tiled map allowing the user to design levels of their game easily. MelonJS is compatible with most browsers. MelonJS is still being altered and fix since it is still a project in working progress.

Description of jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript. This description is from the jquery.com.
